sebelum mengakses web : 

1. pastikan server mysql dan apache telah aktif
2. buat database pada phpmyadmin dengan nama kurma
3. jalankan perintah php artisan migrate:fresh --seed pada terminal(power shell)/git bash
4. jika gambar produk tidak muncul hapus folder image pada folder public->storage->image
5. setelah itu jalankan perintah php artisan storage:link terminal(power shell)/git bash 
6. selanjutnya ketik php artisan serve pada terminal(power shell)/git bash
